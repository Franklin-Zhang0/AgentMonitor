# Internationalization (i18n)

Agent Monitor supports 7 languages with a dropdown language selector.

## Switching Languages

Use the language dropdown in the top-right corner of the nav bar to select your preferred language. Your preference is saved to `localStorage` and persists across sessions.

## Supported Languages

| Language | Code | Coverage |
|----------|------|----------|
| English | `en` | Full |
| Chinese (中文) | `zh` | Full |
| Japanese (日本語) | `ja` | Full |
| Korean (한국어) | `ko` | Full |
| Spanish (Español) | `es` | Full |
| French (Français) | `fr` | Full |
| German (Deutsch) | `de` | Full |

## What Gets Translated

- Navigation labels
- Button text and form labels
- Placeholder text
- Slash command descriptions
- Status messages and system output
- Error messages

## Technical Details

- Translations are in `client/src/i18n/translations.ts`
- Uses React Context (`LanguageProvider`) with a custom `useTranslation()` hook
- No external i18n library needed
- Technical terms (CLI flags, model names, provider names) remain untranslated
